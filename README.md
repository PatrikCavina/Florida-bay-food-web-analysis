# Florida-bay-food-web-analysis

La seguente repo contiene l'analisi della food-web **Florida Bay** nel periodo delle piogge.

## Contenuto

All'interno della repository troviamo:
- `florida-food-web.Rmd`: Analisi completa in RMarkdown della food-web Florida Bay
- `florida-food-web.html`: Versione HTML del file RMarkdown
- `florida-food-web_presentation.pdf`: Presentazione in formato pdf dell'analisi
- `florida-food-web_presentation.pptx`: Presentazione in formato originale pptx
